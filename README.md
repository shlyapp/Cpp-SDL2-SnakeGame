# Cpp-SDL2-SnakeGame

Ремейк классической игры "Змейка", созданный на C++ с использованием SDL2.

Функиональность:
- Меню с выбором
- Просмотр рекордов

Правила игры:
- Управление стрелками
- Красное яблоко +1 очко
- Синее яблоко +5 очков
- У карты нет границ
- Игра начиается сначала, если змейка врезалась в себя
